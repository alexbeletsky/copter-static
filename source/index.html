Features

Deploy your javascript applications to cloud just with `grunt seismo:production` or by pushing repo to remote git repository.
Deployment for different environments by different strategies, e.g. `pure-app to nginx` or `pure-app to CDN`.
Enchance your Javascript applications with SEO capabilites without any line of code, just by application config `seismo app config seo:on`

"As Heroku for back-end, as Seismo for front-end"

What is Seismo

Seismo is PaaS (platform as service) specially designed for front-end (static) applications.

Who is Seismo for?

Seismo is used by developers and businesses to host anything from blogs to high loaded applications.

Why use Seismo?

Seismo is seamlessly integrated into existing development workflow and allow to host applications on high performance servers for best user experience.
